# order-predictor

## What is this

**order-predictor** is a web application combining a Python/Flask backend and a React frontend that uses the Google Gemini API to help determine viable “stocks” based on a user’s prior data.

## Features

- Backend server built with Python + Flask
- Frontend built with React.js
- Integration with a GEMINI API to analyze user data and suggest viable stocks/orders
- Modular project structure (separate API and UI folders) for easier maintenance and updates
